# Creatiview

## Setup

> Needs yarn and node

1. Clone/download repo. In termanal, move to your local copy's folder
2. Run `yarn` to install dependencies.
3. Run `yarn start` to serve app on localhost

For a quick component overview, run Storybook
`yarn storybook`
